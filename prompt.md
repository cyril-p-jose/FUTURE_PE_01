Create a modern AI-powered Website Copy Generator web application for local businesses.

The application should help users instantly generate professional, conversion-focused website copy for businesses such as salons, cafes, clinics, coaching centers, agencies, freelancers, and restaurants.

Website Goal

The platform should allow users to:

Enter business details
Choose business type
Select tone/style
Generate ready-to-use website content

The generated copy must feel human, persuasive, clear, and suitable for real business websites.

Core Features
1. Homepage Copy Generator

Generate:

Headline with strong value proposition
Subheadline explaining customer benefit
Short intro/about section
Customer benefit points
Trust-building content

The homepage copy should:

Be concise
Focus on customer pain points
Highlight benefits instead of features
Sound natural and professional
2. Services Page Generator

Generate:

Service names
Short service descriptions
Benefits of each service
“Why choose us” section

The service copy must adapt based on business type:

Salon → friendly and stylish
Cafe → warm and inviting
Clinic → professional and trustworthy
Agency → confident and modern
3. CTA Generator

Generate persuasive CTA sections such as:

Book Appointment
Contact Us
Free Consultation
Order Now
Visit Our Store

CTA copy should:

Encourage action
Create urgency
Build trust
Include local relevance

Examples:

“Book Your Appointment Today”
“Trusted by Families Across Kochi”
“Limited Weekend Slots Available”
4. Tone Adaptation

Allow users to choose:

Friendly
Professional
Modern
Premium
Minimal
Confident

The generated copy should automatically adapt based on selected tone.

User Input Fields

Create a clean form with:

Business Name
Business Type
Location
Target Audience
Services Offered
Unique Selling Point
Preferred Tone

Add a “Generate Website Copy” button.

Generated Output Sections

Display generated copy in organized cards:

Homepage Copy
Services Page Content
CTA Sections
Tone Variations

Each section should have:

Copy button
Edit button
Clean formatting
UI Design Requirements

Design Style:

Modern SaaS dashboard
Minimal and professional
Responsive design
Smooth animations
Soft shadows and rounded cards

Color Palette:

Dark navy + teal accents
OR
White + blue modern startup style

Use:

Hero section
Feature cards
Prompt input panel
Generated output preview
CTA banner
Additional Features

Include:

Loading animation while generating
AI typing effect
Prompt history section
Export to markdown or copy option
Technical Requirements

Build using:

React
Tailwind CSS
Modern component-based structure

Ensure:

Mobile responsive
Fast loading
Clean UX
Professional typography
Final Goal

The final website should feel like a real AI SaaS product that agencies and freelancers can use to generate website copy for local businesses quickly and professionally.
