# CopyNest AI – AI Website Copy Generator for Local Businesses

## Prompt Engineering Internship Project – Task 1

This project was developed as part of **Task 1** of my Prompt Engineering Internship at Future Interns.

CopyNest AI is a modern AI-powered web application that helps local businesses instantly generate professional, persuasive, and conversion-focused website copy.

The platform is designed for businesses such as:
- Salons
- Cafes
- Clinics
- Agencies
- Restaurants
- Freelancers
- Coaching Centers

# Project Objective

The goal of this project is to help businesses create:
- Homepage copy
- Service page descriptions
- CTA sections
- Tone-adapted website content

using AI-powered prompt engineering workflows.

---

# Features

## Homepage Copy Generator
Generate:
- Headlines
- Subheadlines
- Intro/About sections
- Customer benefit points
- Trust-building content

---

## Services Page Generator
Generate:
- Service names
- Service descriptions
- Benefit-focused copy
- “Why Choose Us” sections

Business-specific tone adaptation:
- Salon → Friendly & stylish
- Cafe → Warm & inviting
- Clinic → Professional & trustworthy
- Agency → Modern & confident

---

## CTA Generator
Generate persuasive CTAs such as:
- Book Appointment
- Contact Us
- Free Consultation
- Order Now
- Visit Our Store

---

## Tone Adaptation
Users can select:
- Friendly
- Professional
- Modern
- Premium
- Minimal
- Confident

The generated copy automatically adapts to the selected tone.
